## Hướng dẫn cài đặt

ApiCovidVietNam yêu cầu [Node.js](https://nodejs.org/) v14+ để chạy.

Cài đặt các phụ thuộc và devDependencies và khởi động máy chủ.

```sh
cd apicovidVietNam

node index.js
```

Đối với môi trường sản xuất ...

```sh
npm install axios
npm install cheerio
npm install express
```

Xem trang web tại: http://localhost:8080/api/

```sh
127.0.0.1:8080
```
